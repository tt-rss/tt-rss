# Tiny Tiny RSS 用户指南

Tiny Tiny RSS (tt-rss) 是一个免费、灵活、开源的基于 Web 的新闻订阅源（RSS/Atom 等）阅读器和聚合器。

## 快速入门

### 1. 安装与配置

请参考 [安装指南](https://tt-rss.org/docs/Installation-Guide.html) 进行部署。

**使用 Docker 快速启动：**

```bash
# 复制环境变量配置
cp .env.example .env

# 启动所有服务
docker compose up -d

# 查看服务状态
docker compose ps
```

### 2. 首次使用

1. **访问界面**：打开浏览器访问 `http://localhost:8280`（或您配置的地址）
2. **登录账户**：使用管理员账户登录
3. **订阅源**：点击订阅按钮，输入 RSS/Atom 订阅地址
4. **浏览文章**：在左侧边栏选择订阅源，右侧查看文章列表

### 3. 界面布局

```
┌─────────────────────────────────────────────────────────────┐
│  工具栏 (搜索、刷新、操作菜单)                                │
├──────────┬──────────────────────────────────────────────────┤
│          │  文章列表                                         │
│  订阅源   │  - 文章标题 1                                     │
│  列表     │  - 文章标题 2                                     │
│          │  - 文章标题 3                                     │
│          ├──────────────────────────────────────────────────┤
│          │  文章阅读区（三栏模式）                            │
│          │  文章内容显示在此区域                              │
└──────────┴──────────────────────────────────────────────────┘
```

**视图模式：**
- **三栏模式**：订阅源列表 + 文章列表 + 文章阅读区
- **组合模式**：订阅源列表 + 文章列表（点击展开文章）

## 核心功能

### 订阅源管理

#### 添加订阅源

1. 点击工具栏的 **订阅** 按钮
2. 输入 RSS/Atom 订阅地址
3. 选择分类（可选）
4. 点击 **订阅** 确认

**快捷键：** `f s`

#### 编辑订阅源

1. 选中要编辑的订阅源
2. 点击 **编辑订阅源** 或按 `f e`
3. 修改名称、URL、分类、更新间隔等
4. 保存更改

#### 标记为已读

- **当前订阅源**：按 `f q` 将当前订阅源标记为已读
- **下方所有**：按 `c p` 将下方所有文章标记为已读
- **上方所有**：按 `c n` 将上方所有文章标记为已读
- **全部订阅源**：按 `Q` 将所有订阅源标记为已读

#### 刷新订阅源

- **刷新当前**：按 `f r` 刷新当前订阅源
- **刷新全部**：点击工具栏刷新按钮

#### 隐藏已读订阅源

按 `f a` 切换显示/隐藏已读的订阅源

### 文章管理

#### 浏览文章

- **下一篇文章**：按 `n` 或 `N`
- **上一篇文章**：按 `p` 或 `P`
- **滚动文章**：使用方向键 `↑` `↓` 或 `Shift+↑` `Shift+↓`

#### 标记文章

| 操作 | 说明 | 快捷键 |
|------|------|--------|
| 星标 | 标记为重要/收藏 | `s` |
| 发布 | 标记为公开分享 | `S` |
| 未读 | 切换未读/已读状态 | `u` |

#### 文章标签

1. 选中文章
2. 按 `T` 打开标签编辑对话框
3. 添加或删除标签
4. 标签可用于分类和筛选文章

#### 文章操作

| 操作 | 快捷键 | 说明 |
|------|--------|------|
| 在新窗口打开 | `o` | 在浏览器新标签页打开文章链接 |
| 发送邮件 | `e` | 通过邮件分享文章（需启用邮件插件） |
| 关闭文章 | `a q` | 关闭/折叠文章（组合模式） |
| 展开/折叠 | `a t` | 切换文章展开状态（组合模式） |
| 全文切换 | `a e` | 使用 Readability 切换全文显示 |

### 搜索功能

#### 搜索文章

1. 按 `/` 打开搜索对话框
2. 输入搜索关键词
3. 按 Enter 执行搜索
4. 按 `\` 取消当前搜索

#### 搜索范围

- 可在搜索时指定搜索范围（标题、内容、标签等）
- 支持高级搜索语法

### 视图切换

#### 组合模式 vs 三栏模式

按 `f C` 在组合模式和三栏模式之间切换

- **组合模式**：适合小屏幕，文章在列表中展开阅读
- **三栏模式**：适合大屏幕，独立的文章阅读区域

#### 网格视图

按 `f G` 切换网格视图，以卡片形式显示文章

#### 宽屏模式

按 `a W` 切换宽屏模式，最大化文章阅读区域

### 批量选择

#### 选择文章

| 操作 | 快捷键 | 说明 |
|------|--------|------|
| 全选 | `a a` | 选择所有文章 |
| 选择未读 | `a u` | 选择所有未读文章 |
| 选择星标 | `a U` | 选择所有星标文章 |
| 选择已发布 | `a p` | 选择所有已发布文章 |
| 反选 | `a i` | 反转当前选择 |
| 取消选择 | `a n` | 取消所有选择 |

#### 批量操作

选中多篇文章后，可以：
- 批量标记为已读/未读
- 批量添加/删除星标
- 批量添加标签
- 批量删除

### 导航快捷方式

#### 快速跳转

| 快捷键 | 目标 |
|--------|------|
| `g a` | 所有文章 |
| `g f` | 最新文章 |
| `g s` | 星标文章 |
| `g p` | 已发布文章 |
| `g r` | 最近阅读 |
| `g P` | 偏好设置 |

#### 订阅源导航

| 快捷键 | 说明 |
|--------|------|
| `k` | 下一个订阅源 |
| `K` | 下一个未读订阅源 |
| `j` | 上一个订阅源 |
| `J` | 上一个未读订阅源 |

### 偏好设置

按 `g P` 访问偏好设置，可配置：

- **常规设置**：语言、时区、文章显示数量
- **订阅源设置**：更新频率、缓存策略
- **界面设置**：主题、视图模式、字体大小
- **快捷键设置**：自定义快捷键
- **插件管理**：启用/禁用插件

## 常见问题

### Q1: 如何备份和恢复数据？

**备份：**
```bash
# 备份数据库
docker compose exec db pg_dump -U postgres -d ttrss > backup.sql

# 备份配置文件
cp config.php config.php.backup
```

**恢复：**
```bash
# 恢复数据库
docker compose exec -T db psql -U postgres -d ttrss < backup.sql
```

### Q2: 订阅源更新失败怎么办？

1. 检查订阅源 URL 是否正确
2. 按 `f D` 调试订阅源更新
3. 检查服务器网络连接
4. 查看后端日志：`docker compose logs backend`

### Q3: 如何导入 OPML 文件？

1. 在偏好设置中找到 **导入/导出**
2. 选择 **导入 OPML**
3. 上传 OPML 文件
4. 确认导入

### Q4: 如何导出订阅源？

1. 在偏好设置中找到 **导入/导出**
2. 选择 **导出 OPML**
3. 下载 OPML 文件

### Q5: 如何更改界面语言？

1. 按 `g P` 进入偏好设置
2. 找到 **常规** 选项卡
3. 选择 **语言**
4. 保存并刷新页面

### Q6: 文章图片无法显示？

可能是以下原因：
- 订阅源使用了相对路径
- 服务器防火墙阻止了外部图片
- 需要在偏好设置中启用 **总是缓存图片**

### Q7: 如何自定义快捷键？

1. 按 `g P` 进入偏好设置
2. 找到 **快捷键** 选项卡
3. 修改对应操作的快捷键
4. 保存设置

### Q8: 如何安装插件？

1. 将插件文件放入 `plugins/` 目录
2. 在偏好设置的 **插件** 选项卡中启用
3. 部分插件需要额外配置

### Q9: 移动端如何使用？

tt-rss 提供响应式界面，可直接在移动浏览器使用。也可使用第三方客户端：
- Android: FeedMe, TTRSS+
- iOS: Reeder, Unread（需支持 tt-rss 后端）

### Q10: 如何重置密码？

```bash
# 通过命令行重置
docker compose exec backend php ./update.php --reset-password=admin
```
